# 德语学习平台 - 访问密码管理

## 当前访问密码

```
german2026
```

## 修改密码

要修改访问密码，编辑文件：`src/pages/Login.jsx`

找到第 5 行：
```javascript
const ACCESS_PASSWORD = "german2026";
```

将 `"german2026"` 改为你想要的新密码，例如：
```javascript
const ACCESS_PASSWORD = "your_new_password";
```

保存后重新构建即可生效。

## 使用说明

1. **首次访问**：用户需要输入密码才能进入
2. **已验证用户**：密码存储在浏览器 localStorage 中，下次自动登录
3. **退出登录**：点击右上角"退出"按钮，清除授权

## 用户数据说明

- 用户的学习进度（单词标记）保存在浏览器 localStorage
- 清理浏览器数据会导致学习进度丢失
- 建议提醒用户定期备份（可以添加导出功能）

## 部署后的使用流程

1. 部署网站到服务器/托管平台
2. 将访问密码发送给付费用户
3. 用户访问网站，输入密码后即可使用
4. 同一设备下次访问无需再次输入

## 安全提示

- 这是简单的前端密码验证，适合小规模使用
- 如需更高安全性，请考虑后端账号系统
- 建议定期更换密码
